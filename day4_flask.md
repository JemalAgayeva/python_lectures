## Setting Up Flask
In order to use Flask, we must set it up in our virtual environment.<br>
<br>

`cd my_environments`<br>
<br>
For Mac:

`source py3Env/bin/activate`<br>
<br>

For Windows(Command Prompt):

`call py3Env/Scripts/activate`<br>
